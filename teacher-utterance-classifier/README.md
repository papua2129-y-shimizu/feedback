# 教師の発話分類アプリ

授業後に教師の発話を入力し、画像の分類表に基づく6分類へルールベースで自動分類する通常Webページです。

## 使い方

1. `index.html` をブラウザで開く。
2. 発話を入力する。
3. 「自動分類」を押す。
4. 候補を確認し、必要なら分類ボタンで修正する。
5. 「確認して保存」を押す。

記録はブラウザの `localStorage` に保存されます。ログイン、通信、クラウド同期は使いません。

## 分類

- 自己の高まり × 深める
- 自己の高まり × 広げる
- 自己の高まり × 簡単
- 仲間との関わり × 深める
- 仲間との関わり × 広げる
- 仲間との関わり × 簡単
